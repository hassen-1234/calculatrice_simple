# Application Calc (Flet)

[English](README_en.md) | [العربية](README.md)

Une application de calculatrice moderne et élégante conçue avec le framework [Flet](https://flet.dev/) en Python.

## Fonctionnalités
- **Arithmétique de base** : Addition, soustraction, multiplication et division.
- **Design Responsive** : Une interface utilisateur propre avec un thème sombre.
- **Multi-plateforme** : peut fonctionner sur Windows, Linux, macOS et être déployée en tant qu'application web.

## Prérequis
- Python 3.9 ou supérieur
- Bibliothèque Flet

## Installation
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/hassen-1234/calculatrice_simple.git
   cd calculatrice_simple
   ```
2. Installez les dépendances :
   ```bash
   pip install flet
   ```

## Utilisation
Pour lancer l'application exécutez :
```bash
flet run
```
