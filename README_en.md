# Calc App (Flet)

[Français](README_fr.md) | [العربية](README.md)

A modern and sleek calculator application built using the [Flet](https://flet.dev/) framework in Python.

## Features
- **Basic Arithmetic**: Addition, subtraction, multiplication, and division.
- **Responsive Design**: A clean, dark-themed user interface.
- **Cross-Platform**: Can run on Windows, Linux, macOS, and be deployed as a web app.

## Prerequisites
- Python 3.9 or higher
- Flet library

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hassen-1234/calculatrice_simple.git
   cd calculatrice_simple
   ```
2. Install dependencies:
   ```bash
   pip install flet
   ```

## Usage
To run the application execute:
```bash
flet run
```
